About

Major blocks of our system

  • Front-end

    This consists of web application user interface. User can enter name of the company whose need for forex services is to be predicted.


  • Bing search engine API

    This block returns the list of URLs obtained after doing a Bing search for the company name


  • Web scrapper

    Web scrapper takes as input the list of urls returned by Bing and in return scraps (downloads) all the text information on these webpages.


  • Natural Language Processing

    The input to NLP model is the text gathered by the web scrapper and output is the list of features extracted from the text. It uses named entity recognition technique of natural language processing.


  • Machine learning model

    Machine learning model takes as input the list of features and gives as ouput the probability of that particular company needing forex services
